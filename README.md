## Overview
This is a simple Streamlit-based quiz bot that asks a few questions about your day and detects your overall emotion based on your responses.

## How to Run

1. Install dependencies:
    pip install -r requirements.txt

2. Train the model:
    python train_emotion_model.py

3. Run the app:
    streamlit run mood_quiz_app.py
